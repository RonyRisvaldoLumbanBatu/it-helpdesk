# IT Helpdesk System

## Deskripsi
Sistem IT Helpdesk untuk pengelolaan tiket dan layanan IT.

## Fitur Saat Ini
- **Pengajuan Perubahan Password Email**: Modul khusus untuk user mengajukan reset atau perubahan password email kantor.

## Teknologi
- **Backend**: PHP
- **Frontend**: HTML/CSS/JS (Vanilla atau Library ringan)
- **Database**: MySQL (Rencana)

## Struktur Folder
- `/public`: File yang diakses publik (index.php, css, js, images)
- `/src`: Logika backend (Controllers, Models, Helpers)
- `/views`: Tampilan halaman (Template HTML/PHP)
- `/config`: Konfigurasi database dan env
